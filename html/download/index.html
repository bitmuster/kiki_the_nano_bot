Download

The current version of kiki is 0.9.2.

It is available as a binary for Windows:

kiki-0.9.2-setup.exe (4 MB)

A slightly older version is available for Mac OS X and in a source package:

kiki-0.9.0.dmg (4.6 MB)

Disk image with binary for Mac OS X.

kiki-0.9.0-src.tgz (3 MB)

The source version includes a ProjectBuilder project file for Mac OS X, a Makefile for Linux and a MS Visual C++ 7 project file for Windows. You have to install a couple of libraries before you can compile it. See the included Readmes for more information.

 

 

Problems

You will need a fast computer with a decent graphics card (hardware accelleration). One possible solution to increase the frame-rate might be to play it in a smaller window.

If you see a white screen with something like the following:

in the lower left corner after starting the program, you have probably no (or a conflicting) Python installation on your machine. Please ask for help in the forum or the mailing list in that case.
