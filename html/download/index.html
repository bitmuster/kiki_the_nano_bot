Download

The current version of kiki is 0.9.0.

It is available as a binary for Windows and in source form:

kiki-0.9.0-win32.zip (3.7 MB)

This is the first Windows binary, please let me know if it worked on your system.

kiki-0.9.0-src.tgz (3 MB)

The source version includes a ProjectBuilder project file for Mac OS X, a Makefile for Linux and a MS Visual C++ 7 project file. You have to install a couple of libraries before you can compile it. See the included Readmes for more information.

The old version (0.8.2) is still available as a binary for Mac OS X:

kiki-0.8.2-dmg.gz (3.3 MB)

 

 

Problems

If you don't have a fast computer you should make sure, that your display is set to thousands of colors, instead of millions. The game might also runs faster in a smaller window.

If you see a white screen with something like the following:

in the lower left corner after starting the program, you have probably no (or a conflicting) Python installation on your machine. Please ask for help in the forum or the mailing list in that case.

Feedback

If you want to make me happy, send feedback to:

monsterkodi@gmx.net

or post a note in the forum or the mailing list.

Currently, I am mainly interested if you were able to compile the game from the source distribution. Irrespective of your success, please let me know on which hardware and operating system you managed (or tried) to compile it.

I also want to know, if you were able to solve the puzzles, how much time you needed to solve them, which levels you liked, which you didn't like, why and why not.
